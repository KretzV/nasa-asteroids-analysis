# NASA Asteroids Analysis

Análise dos asteroides que passaram próximos da Terra nos últimos 30 dias usando a API da NASA.

## Métricas analisadas

- Total de asteroides
- Mais próximo da Terra
- Maior asteroide
- Mais rápido
- Potencialmente perigosos
- Distância média
- Velocidade média

## API utilizada

https://api.nasa.gov/

## Como rodar

```bash
pip install requests
python nasa.py
